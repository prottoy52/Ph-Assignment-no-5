<!-- -----------1----------------- -->
    var is function scoped and can be redeclared and updated.
let and const are block-scoped. let can change value but const cannot.

<!-- ------------2------------------ -->

The spread operator is used to expand elements of an array or object.
It helps copy or merge arrays and objects easily.


<!-- --------------3------------------ -->

map() creates a new array after modifying each element, while filter() returns elements that match a condition.
forEach() only loops through the array but does not return a new array.

<!-- --------------4---------------------- -->
Arrow function is a shorter way to write functions in JavaScript.
It uses the => syntax and is commonly used in modern JavaScript.

<!-- ---------------5------------------------- -->

Template literals are used to create strings with variables easily.
They use backticks ( ) and allow embedding variables with ${}.
